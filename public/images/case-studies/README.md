# Case Study Images

Drop screenshot/export files here with the exact filenames referenced in the
CaseStudyGallery component. See `public/images/SETUP_GUIDE.md` for the full
list of expected filenames per case study and which PDF page each comes from.

Images should be exported at 1200×800px minimum, JPG or WebP, <300KB each.
The site renders them with next/image which handles resizing automatically.
