# Image Setup Guide — josephcarlbriones.com

## How to add the images

Each file below corresponds to a specific page in your PDF case study files.
Take a screenshot of that page (or export from the PDF), crop to the image area,
and save with the exact filename listed.

Recommended export: JPG or WebP, 1200×800px minimum, under 300KB.
The site uses next/image which handles resizing automatically.
